# Dokumentation

- **Setup-Anleitungen:**  
  Schritt-für-Schritt Anweisungen zur Installation und Konfiguration der einzelnen Komponenten.
- **Architektur-Dokumentation:**  
  Beschreibungen der Systemarchitektur und der Interaktion zwischen den Modulen.
