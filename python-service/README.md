# Python Service

Dieses Verzeichnis beinhaltet den Python-Microservice, der zusätzliche Funktionen wie Machine Learning-basierte Prognosen und Analysen bereitstellt.

## Inhalt

- **Ziel:**  
  - Analyse historischer Einsatzdaten
  - Vorhersage von Einsatzzeiten zur Unterstützung der Dienstplanung

- **Technologien:**  
  - Python
  - Flask oder FastAPI (zur Bereitstellung von REST-Endpunkten)
  - Bibliotheken wie scikit-learn oder TensorFlow (optional)

## Erste Schritte

1. Navigiere in diesen Ordner.
2. Richte ein virtuelles Environment ein (z. B. mit `venv`).
3. Installiere die notwendigen Abhängigkeiten (z. B. `pip install -r requirements.txt`).
4. Starte den Service (z. B. `python app.py`).

Weitere Details zur Konfiguration und den ML-Modellen findest du in den Quellcodedateien.
